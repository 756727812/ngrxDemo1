字体图标管理
====

## 流程

1. 导出 svg
2. 上传到 iconfont
3. 下载导出
4. 将导出文件覆盖 src/fonts/sicon
5. 提交

![购物车导入我的项目](font_icon_mgr/购物车添加到项目.jpeg)
